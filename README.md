# Blade RPG

一個以動作戰鬥、角色成長與系統開發為核心的 RPG 專案。

## 專案介紹

Blade RPG 是一個遊戲開發專案，目標是製作一個可持續擴充的 RPG 原型。
目前專案會逐步加入戰鬥系統、角色成長、敵人行為、介面與地圖等功能。

## 功能規劃

- 即時戰鬥系統
- 角色能力與升級
- 敵人與 Boss 戰
- 背包與裝備系統
- 任務或關卡流程
- 遊戲介面優化
